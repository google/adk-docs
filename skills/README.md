# ADK Skills

Development skills for building agents with [Google ADK](https://google.github.io/adk-docs). Install into any coding agent via [`npx skills`](https://github.com/vercel-labs/skills).

## Install

```bash
npx skills add google/adk-docs -y -g
```

## Update

```bash
npx skills check -g
npx skills update -g
```

## Skills

| Skill | Description |
|-------|-------------|
| `adk-dev-guide` | Development lifecycle, code preservation rules, model selection |
| `adk-cheatsheet` | Python API reference — agents, tools, orchestration, callbacks, state |
| `adk-eval-guide` | Evaluation methodology — metrics, evalsets, LLM-as-judge, trajectory scoring |
| `adk-deploy-guide` | Deployment — Agent Engine, Cloud Run, CI/CD, secrets |
| `adk-observability-guide` | Observability — Cloud Trace, Cloud Logging, third-party integrations |
| `adk-scaffold` | Project scaffolding via Agent Starter Pack CLI |
